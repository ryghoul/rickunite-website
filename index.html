FPGA Doodle Jump

Created Doodle Jump on a Nexus A7 100T

FPGA Doodle Jump

Leveraging a Digilent Nexys A7-100T FPGA, I implemented the classic Doodle Jump game entirely in Verilog. I designed a VGA controller to generate 640×480 @ 60 Hz timing and double-buffered sprite rendering for the jumper and moving platforms, and built a hardware physics engine to model gravity, jump dynamics, and collision detection. User input is mapped to push-buttons with dead-zone filtering to ensure smooth left/right movement, while a pseudo-random platform generator based on an LFSR keeps gameplay unpredictable. Onboard clock management divides the 100 MHz system clock into a 25 MHz pixel clock for VGA timing and slower clocks for game logic, and I added simple audio feedback via the onboard speaker for scoring events. This project showcases my expertise in synchronous digital design, finite-state machines, timing-critical interfaces, and resource-efficient hardware implementation.

Microcontrollers Stoplight

Created on a PIC18F4620 using MPLABX

Microcontrollers Stoplight

Using the Microchip PIC18F4620 and MPLAB X IDE, I built a fully autonomous stoplight simulator in C. I mapped the red, yellow, and green LEDs to digital output pins and configured the PIC’s oscillator and configuration bits for an 8 MHz system clock. A hardware timer (Timer0) with interrupt-driven overflow provided precise millisecond timing, driving a finite‐state machine that cycles through red, green, and yellow phases with configurable durations. I also implemented a debounced push-button input for a pedestrian‐crossing request, which temporarily inserted a walk signal into the sequence. Throughout the project, I used MPLAB X’s simulator and on-chip debugger to verify timing, validate state transitions, and optimize code size and performance—demonstrating hands-on expertise in microcontroller configuration, interrupt handling, and embedded systems design.

MagicMirror Smart Mirror

Built a Smart Mirror from scratch using a Raspberry Pi

MagicMirror Smart

I built a fully custom smart mirror using the open-source MagicMirror² framework hosted on GitHub and powered by a Raspberry Pi. On the software side, I cloned the MagicMirror² repo, installed it on Raspbian, and customized modules to display time, weather, calendar events, and headline news. For the hardware, I hand-crafted a wooden frame from raw planks I cut and sanded myself, and fitted a sheet of two-way mirror glass into it. I then stripped down an old monitor to its bare LCD panel and mounted it directly behind the glass, connecting it to the Pi. The result is a seamless, wall-mounted smart mirror that combines embedded Linux, hardware hacking, carpentry, and open-source software customization into a polished, functional installation.

Cafe Sumire

Built a Cafe Website HTML/CSS/JS

Cafe Sumire Website

I designed and built a bespoke, responsive website for Café Sumire—TJ Ige and Nera Castro’s specialty coffee shop—using HTML5, CSS3 and vanilla JavaScript. The site opens with a full-screen video hero overlaid by the café’s tagline in a stylish serif font, and a fixed navbar that smoothly transitions from transparent to solid as you scroll. I crafted a custom color palette of sea green, ivory, charcoal gray, and royal blue accents, and leveraged CSS Grid and Flexbox to showcase the menu: each signature drink appears in a card layout with click-to-open modals for ingredients and tasting notes. An About section tells TJ and Nera’s story with professional photography, while the Contact page features a custom form (powered by Email.js) and an embedded Google Map for easy directions. Mobile-first breakpoints ensure the site looks great on phones, tablets, and desktops alike, and I organized the CSS into modular components for easy future updates. The result is a polished, brand-aligned web presence that highlights Café Sumire’s atmosphere and makes it effortless for customers to explore the menu and get in touch.

Cafe Sumire Ticketing System

Developed a full-stack ticketing system

Cafe Sumire Ticketing System

I developed a full-stack ticketing platform for Café Sumire using Node.js on the backend, Docker containers for consistent environments, and Render.com for seamless cloud deployment. The system exposes a RESTful API for creating, updating, and tracking customer tickets, while a responsive HTML/CSS/JavaScript frontend gives café staff an intuitive interface to manage orders in real time. I orchestrated services with Docker Compose, configured network routing and secure HTTPS endpoints, and tuned firewall rules to ensure reliable, secure communication between microservices and the web client. This solution streamlined the café’s order workflow, reduced service bottlenecks, and highlights my expertise in backend and frontend development, containerization, cloud deployment, and network configuration. The website is located at https://cafe-sumire-ticketing.onrender.com/barista.html
